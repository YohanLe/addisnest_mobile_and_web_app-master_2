Image Uploader Component Test

This page provides instructions for testing the newly created image uploader component. The component has been integrated into the main application and can be accessed via the demo route.

Component Features

  • Support for both image files (PNG, JPEG, JPG) and PDF documents
  • Drag and drop functionality for easy file uploading
  • File validation (type, size, maximum count)
  • Visual previews for uploaded files
  • Error handling with visual feedback
  • Responsive design that works on all screen sizes
  • Accessibility features

How to Test

The image uploader component demo can be accessed through the following route:

/image-uploader-demo

You can test the component functionality with the following steps:

  1. Navigate to the demo page
  2. Try uploading image files by clicking on the uploader area
  3. Try dragging and dropping files onto the uploader area
  4. Test file validation by attempting to upload very large files or unsupported file types
  5. Switch between image and PDF tabs to test different file type handling
  6. Review how file previews are displayed
  7. Test the remove functionality by clicking the remove button on uploaded files
Go to Demo Page

Note: The demo page shows both the image uploader and PDF uploader variants, allowing you to test both functionalities.
